## 🚀 Featured Projects

### 🌾 [Mavuno](https://github.com/charity254/mavuno-backend) — Offline-First PWA for Kenyan Farmers
> Go · PostgreSQL · Supabase · Service Workers · IndexedDB · GitHub Actions · Render

Production-grade Progressive Web Application for rural smallholder farmers to manage produce, marketplace listings, and supply agreements — **fully functional without internet**.

- Layered Go REST API (handlers → services → repositories) with JWT auth & RBAC middleware
- Custom offline sync queue with **exponential backoff retry** and OCC conflict resolution
- 15 pages, 15+ JS modules, 10 Go services, 8 DB migrations, 5 analytics endpoints
- Built for real constraints: KES formatting, 48px touch targets, mobile-first UX

![Go](https://img.shields.io/badge/Go-00ADD8?style=flat-square&logo=go&logoColor=white)
